# 🟢 Loxone Icon Generator

Générateur d'icônes PNG pour **Loxone** — convertissez n'importe quel SVG en icônes PNG optimisées pour votre installation domotique.

## ✨ Fonctionnalités

- 🔗 **Import SVG par URL** (lien direct)
- 📁 **Import SVG par fichier** (glisser-déposer ou parcourir)
- ✏️ **Coller du code SVG** directement
- 🎨 **Colorisation** : couleur unie, dégradé ou couleurs d'origine
- 🟢 **Présets couleurs Loxone** (vert, blanc, noir, etc.)
- 📐 **Tailles multiples** : 24, 36, 48, 64, 96, 128, 256, 512 px
- 📦 **Export ZIP** avec toutes les tailles en un clic
- 🏷️ **Nommage automatique** au format Loxone (`icone_48x48.png`)
- 🕓 **Historique** des icônes générées

## 🚀 Utilisation

Ouvrir directement dans le navigateur : `index.html`

## 🛠️ Technologies

HTML5 · CSS3 · JavaScript Vanilla · Canvas API · JSZip
